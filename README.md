# Milo (Not for production...for the moment !)
## Prerequisites
- Linux / macOS environment
- Docker

## Install
- git clone https://github.com/razzium/milo.git
- cd milo
- sh milo.sh
- [SERVER_URL] : 9888 (defaut credentials : admin@admin.com/password)

## Features
  - Webserver : Apache || NGINX
  - Php
  - MariaDB / MySQL
  - phpMyAdmin
  - SFTP
